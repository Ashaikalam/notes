Manual Payment Voucher (MPV) – SAP F-53

Step 1: Access Transaction Code F-53

Click the MPV button from your SAP menu list, or type F-53 in the command field and press Enter.

Step 2: Enter Company Code and Currency

Fill in the Company Code and choose the relevant Currency.

Step 3: Enter Document and Posting Date

Use the same date

Step 4: Select Bank Account

Input the Bank GL Account (e.g., 110100) and any required house bank or account ID.

Step 5: Enter Amount

Fill in the amount to be paid.

Step 6: Process Open Items

Click Process Open Items → Select the invoice(s) you want to clear → Tick the checkbox.

Step 7: Add Reference & Remarks

Enter Reference Number, Document Text, and additional notes (e.g., Cheque No or Payment Reason).

Step 8: Simulate

Click Simulate to preview the double entries and ensure that Debit and Credit match.

Step 9: Post Document

If correct, click Post. SAP will return a Document Number (e.g., 1900001234).

Step 10: View/Print Voucher (Optional)

Use T-code FB03 to view the posted document. You may print or export it for filing.
